# Blinq Generated Tests

This project contains Playwright Test runner code generated by Blinq.

## Project Structure

```text
.
├── package.json
├── .npmrc
├── README.md
├── playwright.config.ts
├── config/
│   └── environments.json
├── data/
│   └── test-data.json
├── support/
│   └── bvt.ts
└── tests/
    └── **/*.spec.ts
```

- `tests/` contains generated Playwright Test specs, grouped by Blinq folders and features.
- `support/bvt.ts` configures the Blinq Playwright fixture used by generated specs.
- `config/environments.json` maps Blinq environment names to their base URLs.
- `data/test-data.json` stores generated global and environment-scoped seed test data.

## Install

```bash
npm install
npm run playwright:install
```

## Run Tests

Run the generated Playwright tests:

```bash
npm test
```

Run against a specific Blinq environment:

```bash
BLINQ_ENVIRONMENT=staging npm test
```

Run an example Playwright command against the generated specs:

```bash
npm run test:example
```
